-- Site Fictício de ONG — Projeto Acadêmico --
Este repositório contém um site fictício de uma ONG, desenvolvido como parte de uma atividade da faculdade. O projeto tem como objetivo praticar e demonstrar conhecimentos em desenvolvimento web, utilizando as linguagens HTML, CSS e Java.

 -- Tecnologias Utilizadas --
HTML — responsável pela estrutura das páginas e organização do conteúdo;
CSS — utilizado para a estilização e o design visual do site;
Java — empregado na implementação de funcionalidades dinâmicas e na lógica de programação.

-- Objetivo do Projeto --
O site foi criado exclusivamente para fins educacionais, sem fins lucrativos, com o intuito de simular a presença online de uma ONG.
A proposta é representar como uma organização sem fins lucrativos poderia divulgar suas causas, projetos e formas de contribuição através de um site moderno e funcional.
